# Expense Tracker - Full Stack Application

Personal Finance Management System built with Angular and Node.js/PostgreSQL.

## Structure
- **frontend/** - Angular 19 application
- **backend/** - Node.js/Express API with PostgreSQL

## Technologies
- Frontend: Angular 19, TypeScript, Chart.js
- Backend: Node.js, Express, PostgreSQL
- Authentication: JWT

## Quick Start
See README files in frontend and backend folders for setup instructions.
